# solar

## Project setup
Copy `.env.example` to `.env` and fill it out.

Copy `secret.json.example` to `secret.json`

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
serve -s dist
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
